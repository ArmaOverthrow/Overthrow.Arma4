\doxysubsection{Detailed Description}
Handles the spawning and respawning logic for players within the Overthrow game mode. Extends the base EPF respawn system to integrate with Overthrow-\/specific player data and spawning rules. Should be attached to the \doxylink{class_o_v_t___overthrow_game_mode}{OVT\+\_\+\+Overthrow\+Game\+Mode} entity.
